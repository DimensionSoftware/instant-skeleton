Given a route name and parameters, return a URL path. (Based on Camping's R function)

Parameters:

  • name must be a String.
    (route name)

    • ...args must be a String.
      (varargs for each path variable of route)

Returns a String
(path)

export r = (name, ...args) -> route = list |> find -> it.0 is name if not route throw new Error("Route '#name' not found.") fold ((m, i) -> m.replace /:(\w+)/, i), route.1, args

Given a route name, route.list index and parameters, return a URL path.

Parameters:

  • name must be a String.
    (route name)

  • n must be a String.
    (index in route matches)

    • ...args must be a String.
      (varargs for each path variable of route)

Returns a String
(path)

export rn = (name, n, ...args) -> route = (list |> filter -> it.0 is name)[n] if not route throw new Error("Route '#name' not found.") fold ((m, i) -> m.replace /:(\w+)/, i), route.1, args
